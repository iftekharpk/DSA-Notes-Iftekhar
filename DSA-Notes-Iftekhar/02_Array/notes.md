An array is a collection of elements stored in contiguous memory locations.

**Example:**
```cpp

```

## ðŸ”¹ Important Operations
| Operation | Time Complexity | Description |
|:-----------|:----------------|:-------------|

| Search | O(n) | Linear search |
| Sort | O(n log n) | Merge/Quick sort |
